# Hostel Attendance System

## Description
This is a Java Maven project developed using Apache NetBeans.  
It is used to manage hostel attendance and student records.

## Features
- Student management
- Attendance tracking
- Dashboard for warden/head girl
- Report generation

## Requirements
- Java JDK
- Apache NetBeans
- XAMPP (MySQL)

## Setup Instructions
1. Import project into NetBeans
2. Start XAMPP (Apache + MySQL)
3. Open phpMyAdmin
4. Create a database (e.g., hostel_db)
5. Import the hostel_db.sql file
6. Run the project

## Database
- Database Name: hostel_db
- Default Username: root
- Password: (leave empty for XAMPP)
